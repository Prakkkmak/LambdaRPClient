Menu d'aide

(Utilisez 'Echap' pour quitter)




/perm add [player] [perm]
Permet d'ajouter une permission à un joueur précis

/perm suppr [player] [perm]
Permet de supprimer une permission à un joueur précis

/orga perm [idOrga] [perm]
Permet d'ajouter une permission à une organisation précise

/orga perm remove [idOrga] [perm]
Permet de supprimer une permission à une organisation précise

/orga perm liste [idOrga]
Permet de check la liste des permissions d'une organisation précise





/inta skin [nombre]
Permet de changer le skin d'un intérieur [1-1007]

/zonea creer [nom] [range] [type]
Permet de créer une zone avec un nom, une portée en spécifiant le type de la zone

/zonea check
Permet de vérifier les paramètres d'une zone dans laquel on se trouve

/zonea liste
Permet d'afficher la liste de toutes les zones créées sur le serveur

/zonea range
Permet de modifier la portée la portée de la zone où on se trouve

/zonea nom [nom]
Permet de modifier le nom de la zone où on se trouve

/zonea suppr
Permet de supprimer la zone où on se trouve

/zonea prix [nombre]
Permet de mettre en vente une zone où on se trouve (-1 = Pas en vente ; +0 = En vente )

/obja creer [obj] [nombre]
Permet de se give un objet

/obja clearinv [player]
Permet de supprimer entièrement l'inventaire d'un joueur

/tradea add [objid] [prixachat] [prixvente]
Permet d'ajouter un objet dans une zone import/export où l'on se trouve

/va creer [vihivle]
Permet de créer un véhicule ( Les clés du véhicule doivent ensuite être give au joueur manuellement )

/va couleur [coul1] [coul2]
Permet de changer la couleur du véhicule dans lequel on se trouve

/va supprimer
Permet de supprimer un véhicule dans lequel on se trouve

/orga creer [leader] [nom]
Permet de créer une organisation en spécifiant qui est le leader

/ban [player]
Permet de ban un joueur (Spécifier le nom entier du Joueur ) même si celui-ci est déco

/unban [player]
Permet de déban un joueur (Spécifier le nom entier du Joueur) même si celui-ci est déco

/moneya set [player] [amout]
Permet de fixer l'argent du joueur à une somme exacte

/moneya add [player] [amout]
Permet d'ajouter de l'argent à un joueur

/moneya rem [player] [amout]
Permet de retirer de l'argent à un joueur




/orga liste
Permet de voir la liste des organisations créées

/tp [x] [y] [z]
Permet de se téléporter à une position exacte (/pos pour connaitre sa position)

/goto [player]
Permet se téléporter à un joueur

/gethere [player]
Permet de téléporter à joueur à soi

/kick [player]
Permet de kick un joueur du serveur en spécifiant la raison

/ban [player] [time]
Permet de ban un joueur temporairement

/freeze [player]
Permet de freeze un joueur

/unfreeze [player]
Permet de unfreeze un joueur

/kill [player]
Permet de tuer un joueur, qu'importe sont endroit sur la map
